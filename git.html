Les commandes GIT

Commandes Description
git init Créer un repository local
git clone Copier un repository
git add Ajouter un fichier dans le prochain commit
git commit Sauvegarder dans le dépôt en local
git push Publier les changements sur le repository original
git branch Afficher toutes les branches
git checkout Changer de branche
git merge Fusionner la branche précisée sur la branche en cours
git log Montrer tous les commits sur la branche en cours
git status Afficher les fichiers pris en compte par le commit (vert = ajouté et rouge = non ajoutés)
git remote add [nom] [url] Créer une connexion avec le repository d'origine
git pull Télécharger les derniers changements sur le repository d'origin avec fusion
